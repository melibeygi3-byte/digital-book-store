# توضیح کلی پروژه

فروشگاه کتاب دیجیتال چیه؟


فروشگاه کتاب دیجیتال یک پلتفرم آنلاین هست که کاربران می‌تونن کتاب‌ها رو به صورت الکترونیکی (PDF، EPUB، صوتی و …) بخرن، دانلود کنن یا حتی داخل همون سایت/اپلیکیشن مطالعه کنن.
این فروشگاه‌ها معمولاً مثل یک کتاب‌فروشی واقعی عمل می‌کنن، فقط همه‌چیز آنلاین و بدون نسخه فیزیکی انجام میشه

# اهداف پروژه
1) دسترسی راحت و سریع به کتاب‌ها.

2) کاهش هزینه‌ها (برای خریدار و فروشنده)
   
3) گسترش فرهنگ مطالعه
